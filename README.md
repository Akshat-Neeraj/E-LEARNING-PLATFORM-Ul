## 📋 Overview

E-Learning Platform UI is a frontend design project showcasing a clean, intuitive interface for online learning platforms. Built to explore modern web design principles, responsive layouts, and user-centered design.

**Project Goals:**
- Learn modern UI/UX design principles
- Build responsive layouts for educational platforms
- Explore accessibility best practices
- Create an intuitive user interface

## ✨ Features

### Current Features
- 🎨 **Clean Design** - Modern, professional interface
- 📱 **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
- 🎯 **Course Cards** - Attractive course display layout
- 🔍 **Search Functionality** - Easy course discovery
- 📊 **Dashboard Layout** - Student progress overview
- ♿ **Accessible** - WCAG compliant design

### Planned Features

## 🛠️ Tech Stack

**Frontend:**

## 📦 Installation

### Prerequisites
- Modern web browser
- No dependencies required!

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Akshat-Neeraj/E-LEARNING-PLATFORM-Ul.git
   cd E-LEARNING-PLATFORM-Ul
   ```

2. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   open index.html
   ```

## 🚀 Usage

### Viewing the Platform
- Open `index.html` in any modern web browser
- Navigate through different sections using the menu
- Resize browser to see responsive design in action

### Key Sections
- **Home Page** - Landing page with featured courses
- **Course Catalog** - Browse all available courses
- **Dashboard** - Student progress and enrolled courses
- **Course Detail** - Individual course information
